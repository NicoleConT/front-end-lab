## start

``` shell
# 本地开发
yarn docs:dev

# 构建文档
yarn docs:build
```

## 开发分支

| branch  | description                                                            |
| ------- | ---------------------------------------------------------------------- |
| master  | 主/文档分支。编写笔记在此分支进行更新                                  |
| develop | 开发分支。项目构建类先在开发分支进行，开发完毕后再合并至 master 分支中 |
